Core package
3

Guide, maintenance, and audit skills designed to work together while staying reusable across vaults.

Method constraints

Noise filtering, stable page roles, milestone-only logs, and profile-based configuration instead of hardcoded local structure.

What’s inside

The kit is designed for people who want a vault that stays navigable and durable over time instead of turning into a long pile of project notes.

knowledge-base-kit-guide

Explains installation, profile setup, and which skill to use first when the user is new to the kit.

knowledge-base-maintenance

Writes durable conclusions into the right page role, filters process noise, and syncs content plus navigation surfaces.

knowledge-base-audit

Checks metadata, dead links, orphan pages, boundary drift, navigation coverage, and noise regression.

How it flows

The workflow is intentionally simple: configure once, then run maintenance and audit as separate loops.

01InstallCopy the three skill folders into your AI platform’s skills/ directory.
02ProfileFill the vault profile template with your vault root, root pages, areas, and metadata rules.
03MaintainUse maintenance to write durable conclusions and reusable troubleshooting knowledge.
04AuditRun audit after bigger changes or on a schedule to catch drift, dead links, and noise regression.
