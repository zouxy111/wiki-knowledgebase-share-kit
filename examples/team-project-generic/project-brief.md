# Project Brief

- Project name: Internal Tool Onboarding Pack
- Current scope status: approved
- Project owner / requester: Example Ops Lead
- Last updated: 2026-04-16

## Goal
把内部工具 onboarding 资料整理成一个可交接的“事实清单 + 新人手册”。

## Success criteria
- 技术事实与操作步骤不冲突
- 新人读手册可以独立完成第一次配置
- 文档边界清楚：事实层和表达层分开

## In scope
- 收集现有安装与使用事实
- 生成新人版 onboarding 手册

## Out of scope
- 改工具本身
- 补做不存在的功能说明

## Constraints
- 现有资料比较零散
- 只有 2 位成员可投入

## Deliverables
- 技术事实清单
- 对外可读 onboarding 手册
