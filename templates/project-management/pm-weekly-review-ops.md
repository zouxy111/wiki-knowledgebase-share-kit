# PM Weekly Review Ops

## 每周 review checklist
- [ ] 本周 active / blocked / done / parked 是否已更新
- [ ] Portfolio board 是否仍反映真实状态
- [ ] Personal execution board 的 next actions 是否已刷新
- [ ] 新 blocker 是否进入 risk register
- [ ] 新决策是否进入 decision register
- [ ] 需要多人协同的事项是否已转入 team coordination
- [ ] 已形成稳定知识的事项是否已转给 maintenance
- [ ] 需要 closeout 审计的项目是否已转 delivery audit

## Review 节奏建议
- 周初：刷新本周推进板
- 周中：更新 blocker / risk / dependency
- 周末：做 distill 和 routing
