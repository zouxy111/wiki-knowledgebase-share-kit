# PM Delivery Gates

## Gate states
- `blocked`：关键证据 / 决策 / 回写缺失
- `ready`：主体闭环成立，可进入人工批准 / handoff
- `greenlight pending`：等待批准，不要提前宣告通过

## Ready checklist
- [ ] 成功标准明确
- [ ] 主要交付物可指向
- [ ] 关键决策已记录
- [ ] 需要回写的业务结论已回原 area
- [ ] handoff 所需材料齐全

## Blocked examples
- 只说 done，没有证据
- task board 标 done，但业务 area 还没回写
- 决策没有 rationale
- handoff 信息不足
