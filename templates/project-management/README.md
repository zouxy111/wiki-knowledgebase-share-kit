# Project Management Template Family

> 用途：给核心 share kit 提供一个**可选的 PM 管理层 area 模板族**。
> 这组模板用于 `project-management` area，不替代业务 area，也不替代 `governance`。

## 什么时候复制这组模板
当你明确需要这些能力时再复制：
- 项目 intake
- 里程碑 / blocker / dependency 管理
- portfolio 视图
- 个人执行板
- delivery gates
- 项目复盘 / 决策 register

如果你只是做普通知识库维护：
- 可以先不启用这组模板
- 继续使用原本的 ingest / maintenance / audit 主线

## 推荐放置位置
通常放在你的 vault `pages/` 目录下，并把它们的 `area` 设为：
- `project-management`

## 模板列表
- `project-project-management-overview.md`
- `pm-portfolio-board.md`
- `pm-operating-model.md`
- `pm-personal-execution-board.md`
- `pm-weekly-review-ops.md`
- `pm-delivery-gates.md`
- `pm-risk-register.md`
- `pm-decision-register.md`

## 与其他主线的边界
- 业务事实 → 原业务 area
- 多人共享项目协调 → `knowledge-base-team-coordination`
- 交付闭环 / greenlight 审计 → `knowledge-base-delivery-audit`
- 稳定结论回写 → `knowledge-base-maintenance`
