# Linking Strategy for Ingested Page Groups

## Goal
Ensure imported pages form a real knowledge structure, not a pile of isolated notes.

## Minimum link set
Every imported page group should consider:
- source overview link
- parent link
- child list
- prev / next navigation for chapter-like sequences
- related links for concept overlap

## Recommended three-layer pattern

### 1. Overview page
Contains:
- source identity
- import scope
- table of contents
- major chapter links
- glossary or key-topic links

### 2. Chapter page
Contains:
- chapter summary
- key concepts
- child topic links
- prev / next chapter navigation
- backlink to source overview

### 3. Topic page
Contains:
- normalized concept or method
- source attribution
- related links to neighbor topics and chapters

## Related-link suggestions
A related-link suggestion is useful when:
- two pages share multiple stable keywords
- one page should naturally mention the other during retrieval
- the link improves navigation instead of adding noise

Do not keep all suggested links.
Prefer a smaller, meaningful set.
