# Delivery Audit Checklist

## Checklist
- 成功标准是否明确
- 当前交付物是否与成功标准对齐
- 关键证据是否可指向
- 关键决策是否记录到 decision register / equivalent file
- 需要回写业务 area 的内容是否已回写
- handoff 所需信息是否足够让下一位接手
- 当前状态应是 `ready` 还是 `blocked`

## Typical findings
- 只说“做完了”，但没有证据
- 有 deliverable，但没有 decision rationale
- 任务板显示 done，但业务 area 未回写
- 已经 closeout，但 handoff checklist 为空
