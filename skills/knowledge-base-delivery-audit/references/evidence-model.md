# Evidence Model

## Delivery evidence can include
- 已完成文档 / 页面 / 代码 / 数据输出
- 已批准的 assignment / task board 状态
- 已记录的关键决策
- 已回写的业务知识页
- 已准备好的 handoff note / gate note

## What is not enough by itself
- 口头说“已完成”
- 聊天里一句“done”
- 没有路径、没有页面、没有输出物的描述
- 还停留在 `draft` 的草案

## Rule
如果证据无法指向具体 artifact，就不能把它当成“已充分交付”。
