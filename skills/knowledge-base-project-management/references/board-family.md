# Board Family

## Template family
推荐模板位于：
- `templates/project-management/project-project-management-overview.md`
- `templates/project-management/pm-portfolio-board.md`
- `templates/project-management/pm-operating-model.md`
- `templates/project-management/pm-personal-execution-board.md`
- `templates/project-management/pm-weekly-review-ops.md`
- `templates/project-management/pm-delivery-gates.md`
- `templates/project-management/pm-risk-register.md`
- `templates/project-management/pm-decision-register.md`

## Suggested page-role mapping
- `project-project-management-overview.md` → `type: project`
- `pm-operating-model.md` → `type: knowledge`
- `pm-portfolio-board.md` → `type: task`
- `pm-personal-execution-board.md` → `type: task`
- `pm-weekly-review-ops.md` → `type: ops`
- `pm-delivery-gates.md` → `type: ops`
- `pm-risk-register.md` → `type: task`
- `pm-decision-register.md` → `type: task`

## What goes where
- 稳定方法 → `pm-operating-model.md`
- 组合状态 / 项目阶段 → `pm-portfolio-board.md`
- 本周动作 / blocker / next action → `pm-personal-execution-board.md`
- 风险与依赖 → `pm-risk-register.md`
- 重要决策及状态 → `pm-decision-register.md`
