# Routing Boundaries

## Stay in project-management when
- 还在项目 owner 视角做 intake / 拆解 / 排优先级
- 还在整理 milestones / blockers / next actions
- 还在维护 portfolio / personal execution board

## Route to team coordination when
- 需要 2 人及以上配合
- 需要差异化问卷
- 需要 alignment / assignment / follow-up
- 需要共享项目目录作为事实源

## Route to maintenance when
- 已形成稳定业务结论
- 需要把结果回写到原业务 area
- 需要更新 root page / index / log 等导航面

## Route to delivery audit when
- 项目声称“完成”
- 需要判断是否 ready / blocked / greenlight
- 需要检查证据、决策、回写、handoff 是否闭环
